# SiteAverager
Averages colour on a given site/query by detecting images on the DOM
## Requirements
  - Python 3
  - Node 8.9
  - Typescript 2.7

Python 3 must be in path for python-shell to execute average.py 

## Usage

Update packages with:

`npm install`

Get typescript:

`npm install typescript -g`

and compile with

`tsc`
